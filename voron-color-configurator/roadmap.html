Roadmap for future updates and fixes

Adding Voron V2.4 for color configuring.

Adding Voron V0.1 for color configuring.

Adding Voron Trident for color configuring.

Adding Voron Switchwire for color configuring.

Adding Voron Legacy for color configuring.

Custom colored frames
Being able to color frames has been suggested by one of my testers, which I of course will add in the near future, since it will bring another layer of customization to the printers. Making each design potentially even more unique.

Downloadble configuration files
Being able to download all the STL files, ordered in folders in relation to the level of configurations chosen. For example Voron V2.4 with primary color and 6 accent colors, will have a folder for the primary color and 6 other folders for the accent colors.

Adding more colors to the color picker
A suggestion from one of my test users has shown interest in adding more colors from other manufacturers as well, beside eSUN. Another popular choice of plastic for the Voron Printers are, ABS colors from Keene Village Plastics (KVP), which I will add as soon as possible.

Mozilla Firefox - Fixing black screen in canvas
One of my test-users pointed out to me that when using Mozilla Firefox, the canvas where the 3D model of the printers is displayed, it will show a black screen at submission. If you have this problem, I recommend using a Chromium based browser like Edge or Chrome, for now.

Save configuration after submission
In the current version of the website, there is a problem after having submitted the config, where the site resets what has been set. This needs to be fixed.

Dark Mode
In the future, we will explore adding the option to choose between light mode and dark mode. This is however not currently on high priority.
