Roadmap for future updates and fixes

Adding Voron Printers for color configuring.

Mozilla Firefox - Fixing black screen in canvas
One of my test-users pointed out to me that when using Mozilla Firefox, the canvas where the 3D model of the printers is displayed, it will show a black screen at submission. If you have this problem, I recommend using a Chromium based browser like Edge or Chrome, for now.

Save configuration after submission
In the current version of the website, there is a problem after having submitted the config, where the site resets what has been set. This needs to be fixed.

Dark Mode
In the future, we will explore adding the option to choose between light mode and dark mode. This is however not currently on high priority.

About

Our mission with this tool is to make it more convenient for the user to visualize what they want their 3D printer color theme to look like. Users can now create and visualize very unique looking builds, where currently up to 8 different groups of parts can be colored (this holds for Voron printers).
